# Ghost Plugins for Claude Code

AI-powered browser automation. Auto-generates MCP tools for any website — 26x faster than screenshot-based approaches.

## Install

```bash
/plugin marketplace add ajsai47/ghost-plugins
/plugin install ghost@ghost-plugins
```

## What is Ghost?

Ghost watches what you browse and generates structured tool definitions for any website. Instead of expensive screenshot → vision model loops, Claude Code calls purpose-built tools that interact with the DOM directly.

- **Instant tool generation** — visit a site, get tools in seconds
- **26x faster** than screenshot-based browser automation
- **Works with any website** — news, ecommerce, social, SaaS, APIs
- **Local-first** — tool registry stored in SQLite on your machine

## Requirements

- Chrome browser with Ghost extension installed
- Node.js 20+
